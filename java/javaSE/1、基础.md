### 一、修饰符

- 什么是修饰符

  定义类、方法或变量的访问权限

- 类型

  - 访问修饰符
    - public
    - protected
    - default
    - private
  - 非访问修饰符

- 关键

  - 访问修饰符级别

    | 修饰符    | 当前类 | 同一包类 | 不同包子类 | 不同包子类 |
    | --------- | ------ | -------- | ---------- | ---------- |
    | public    | Y      | Y        | Y          | Y          |
    | protected | Y      | Y        | Y          | N          |
    | default   | Y      | Y        | N          | N          |
    | private   | Y      | N        | N          | N          |

  - 非访问修饰符

    如static、final、abstract、synchronized

### 二、数据类型

  - float
  - double
  - boolean
  - char
- 引用数据类型
  - 类
  - 接口
  - 数组
- 数组类型初始值
  - byte、short、 int、 long 默认初始值：0
  - float、double默认初始值：0.0
  - char字符类型默认初始值:  '\u0000'
  - boolean布尔类型:  false
  - 数组等引用类型:  null
