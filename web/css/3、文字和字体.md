文本属性

| 属性            | 说明                           |
| --------------- | ------------------------------ |
| text-decoration | 下划线、删除线、顶划线         |
| text-transform  | 文本大小写                     |
| font-variant    | 将英文文本转换为“小型"大写字母 |
| text-indent     | 文本首行缩进                   |
| text-align      | 文本水平对齐                   |
| vertical-align  | 文本垂直对齐                   |
| line-height     | 行高                           |
| letter-spacing  | 字距                           |
| word-spacing    | 词距                           |
