​		判断逻辑运算符是否还需要计算下去，短路||右边为true则表达式为true,不需要计算下去，返回当前转型的值

```
// 短路||  如果左边为真则返回左边值，不计算右边，短路
console.log(5 || 3) // 5
console.log("" || 4) // 4
// 短路&&  如果左边为false则返回左边值，不计算右边，短路
console.log("" && 5) // ""
console.log(45 && 233) // 233
```
