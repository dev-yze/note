git初始化使用教程

- git init
- git config user.name ""
- git config user.email ""

```c
18856@DESKTOP-EBLB3S0 MINGW64 /f
$ cd note
18856@DESKTOP-EBLB3S0 MINGW64 /f/note
$ ls
// 初始化
18856@DESKTOP-EBLB3S0 MINGW64 /f/note
$ git init
Initialized empty Git repository in F:/note/.git/
设置项目对应用户名和密码
18856@DESKTOP-EBLB3S0 MINGW64 /f/note (master)
$ git config user.name "develop-yang"
18856@DESKTOP-EBLB3S0 MINGW64 /f/note (master)
$ git config user.email "18856235862@163.com"
// 创建一个test.text文件
18856@DESKTOP-EBLB3S0 MINGW64 /f/note (master)
$ touch test.text
18856@DESKTOP-EBLB3S0 MINGW64 /f/note (master)
$ ls
test.text
// 提交
18856@DESKTOP-EBLB3S0 MINGW64 /f/note (master)
$ git add --a
warning: LF will be replaced by CRLF in test.text.
The file will have its original line endings in your working directory
18856@DESKTOP-EBLB3S0 MINGW64 /f/note (master)
$ git commit -m "test"

 // 连接服务器github
```
