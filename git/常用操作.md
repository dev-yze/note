三、创建合并分支

##创建分支          `git branch dev`

##切换分支		`git chekout dev`

##创建并切换分支      `git checkout -b dev`



##返回master分支      `git checkout master`

##将dev分支合并到master              `git merge dev`

##删除分支										`git branch -d dev`

###使用switch切换分支

##创建并切换到新分支			`git switch -c dev`

##切换到已有分支					`git switch master`
