### git管理多个账号

#### 1、生成密钥，产生默认文件id_rsa、id_rsa.pub

`cd ~/.ssh`

`ssh-keygen -t rsa -C "xxxxxx@xxx.com"`

多个账号可生成时命名如 id-rsa_163，产生对应两个文件

#### 2、将id_rsa*.pub中内容复制到github设置中的SSH and GPG keys中，new SSH key

#### 3、配置config

`touch config`

```
# 163
Host 163				##命名
   HostName github.com    ###网站名称，域名,ip
   User develop-yang	 ###登录密码
   IdentityFile ~/.ssh/id_rsa_163  ##3身份文件

# zhuolu
Host zl
   HostName 47.99.192.207
   User yangzhangen
   IdentityFile ~/.ssh/id_rsa
# google
Host google
   HostName github.com
   User yangzhangen
   IdentityFile ~/.ssh/id_rsa_google

```

#### 4、将私钥交给ssh-agent代管

`ssh-agent bash`

`ssh-add id_rsa_163`

`ssh-add id_rsa_google`

#### 4、取消全局命名

`git config --global --unset user.name`
`git config --global --unset user.email`
